### File System.
* If a file with same name or folder exits it will not do anything.
* Always use try/catch block while working with files.
* To write and read files in Java we use:
  * **Formatter**.
  * **FileWriter**.
  * FileReader.
  * BufferedReader.
  * Files.
  * **Scanner**.
  * FileInputStream.
  * BufferedWriter.
  * FileOutputStream.
